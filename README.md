# WordPress Snippets

A curated collection of **WordPress code snippets** for developers and site owners.

## Snippets Included

- Disable XML-RPC
- Change login URL
- Limit login attempts
- Disable emojis
- Custom functions for themes

## Usage

Copy the snippets into your theme’s `functions.php` or a custom plugin.
